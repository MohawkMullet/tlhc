Turbo Lister

Turbo Lister is a free, easy-to-use listing tool that helps you create professional-looking listings, and upload and edit items in bulk. To get started, download the software from the Turbo Lister page.

To see and compare other selling tools, visit the Selling Tools feature comparison table.

Turbo Lister features

With Turbo Lister you can:

  • Create professional-looking listings using a design editor and templates—no HTML required

  • Edit and create new listings offline and upload them to eBay all at once.

  • List multiple items at the same time.

  • Duplicate existing listings and save listings to reuse them later.

  • Change formats or add item specifics to multiple listings at the same time.

  • Import active or completed item listings from eBay into Turbo Lister.

  • Edit or revise active listings on eBay.

  • Relist and sell similar items on eBay.

System requirements

To get started using Turbo Lister, make sure your system meets the minimum requirements:

  • A computer running Microsoft Windows (Turbo Lister is compatible with Windows 8)

  • Internet Explorer 5.5 or later

  • 500 MB free disk space

  • 256 MB of RAM

Turbo Lister isn't compatible with Macintosh operating systems.

You also need to register with eBay and become a seller. If you want to use Turbo Lister's scheduling feature, place a credit card on file with eBay.

Installing Turbo Lister

To install Turbo Lister:

  1. From the Turbo Lister page, click Download Now.

  2. Save the installation file to your local drive.

  3. Once the file is finished downloading, double-click it.

  4. Follow the onscreen instructions to complete installation.

The installation program places a Turbo Lister shortcut icon on your desktop, on the Quick Launch bar and in the Start menu (All Programs > eBay > eBay Turbo Lister 2). You can start Turbo Lister from any of these places.

Signing in to Turbo Lister

If you are using Turbo Lister 6.5 or higher, you'll be asked to confirm your identity through an authentication process the first time you log in. This authentication is valid for 18 months.

To sign in and confirm your identity:

  1. Open Turbo Lister.

  2. Click Next.

  3. Enter your username.If you list on more than one eBay site, select the check box under your username.

  4. Click Next.

  5. Click Connect Now.

  6. Click Continue.

  7. Sign in to eBay.

  8. Click I agree.

  9. Verify your contact information, and then click Next.

  10. In the Getting Started dialog box, select Create a new item or Synchronize data with eBay, and then click Finish.

Enabling password protection

For increased security, after you've completed authentication and authorization, you can add password protection to your Turbo Lister files.

To enable password protection:

  1. From the Turbo Lister Tools menu, select Options….

  2. In the Options and preferences window, click Advanced Options, and then click Turbo Lister Options.

  3. Click Enable Password Protection.

  4. Click Continue.

  5. Click OK to close the window.

Changing your username in Turbo Lister

If you change your username on eBay, you also need to change your username in Turbo Lister. If you change your password on eBay, you won't need to change it in Turbo Lister.

To change your username in Turbo Lister:

  1. From the Turbo Lister Tools menu, select Options….

  2. In the Options and preferences window, click Personal Information, and then click eBay Account.

  3. Click Change Username.

  4. Enter your new username.

  5. Click OK.

  6. Click OK again to close the window.

Updating Turbo Lister

Turbo Lister will prompt you whenever updates are available. You can also check for updates manually.

To check for updates to Turbo Lister:

  1. From the Turbo Lister Tools menu, select Check for Program Updates.

  2. If there are updates, click Install Now or Install Later.

  3. You'll be prompted to back up your information before the update occurs.

After installing a Turbo Lister update, you'll be asked to confirm your identity.

Uninstalling Turbo Lister

Before you uninstall Turbo Lister, we recommend that you back up your Turbo Lister files in case you want to use Turbo Lister again later.

To back up your Turbo Lister files:

  1. From the Turbo Lister File menu, choose Backup Database.

  2. Click Browse and save your backup file somewhere you can easily find it later.

  3. Click OK.

To uninstall Turbo Lister:

  1. Click the Windows Start button, and then click Control Panel.

  2. Click Add or Remove Programs.

  3. Select Turbo Lister 2 in the list of installed programs and then click Uninstall. Follow the onscreen instructions.

  4. You'll be asked if you want to remove user database files.

  5. Click Change/Remove.

  6. You'll be asked if you want to remove the Turbo Lister database files. You have two 2 choices:

    • Click No if you want to keep the listings and other information that you've entered into Turbo Lister.

    • Click Yes if you want to remove this information. If you click Yes, all your information will be deleted and won't be available in any version of Turbo Lister that you may install in the future.

  7. Click Finish.
