Silk icon set for the jQuery UI

jquery-silk-icons, CSS files for jQuery UI for silk-icons

Just use silk-icon-[icon-name] in place of any normal jQuery UI icon you would use.

The silk icons are in a single sprite so they load fast too.

License stuff

This was originally written for the Leebre UI.

The silk icon set is a awesome CC-BY web-friendly icon set made by Mark James, and can be found here: http://www.famfamfam.com/lab/icons/silk/

This CSS was written by michaelb and released under the GPL 3.0, based on this CSS and image sprite by Rob Richardson, also under the GPL 3.0.
