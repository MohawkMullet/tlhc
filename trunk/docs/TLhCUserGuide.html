Introduction

TLhC was designed as a template companion to ebay’s TurboLister bulk uploader. It provides a form for simple item input and produces the HTML code needed to upload to ebay. Primarily it makes entering auction item details quick and simple. By using templates it also provides a very consistent look and feel to your listings and store.  

TLhC (TurbloLister HTML Cheater) is an application designed to help easily produce consistent looking ebay listings for similar items.  Information is commonly reused. TLhC combines a powerful database and professional looking templates to quickly create HTML code for auction listings.  

It’s an advanced html generator used for quickly creating ebay listing html code to be uploaded by TurboLister.

A listing is created by entering item information into a form. A preview is generated immediately and edits can be made very simply within the same window. The resulting auction listing code is ready to be pasted into TurboLister.

TLhC in its current release is customized for selling 45 RPM record singles. It can be customized to sell anything and example code and documentation is included. TLhC comes with several professional looking templates to keep your Ebay store looking sharp. Documentation for creating your own templates is included.

Features

  1. Config file & Documentation
  2. Research functions built in (Price lookups from multiple sites)
  3. Custom CSS styles included in your listing
  4. Show Latest Listings in item Description
  5. AJAX functions hold form data while switching tabs
  6. Modules easily turned off or on for listings
  7. Show store categories
  8. Include Store Promotion Boxes
  9. Include subscription links (add to my yahoo, rss button, G+, etc)
  10. Reusable comments easily included in listings
  11. Multiple images can be included in listings (no need to pay eBay for extra images)
  12. Multiple templates for different looks to your auctions
  13. Add jQuery plugins. [http://jQuery.com]
  14. Uses jQuery UI
  15. Creates archive of each image and item details
  16. Customization, template design, and contract services available directly from original developer. Auction item listing services also available.
  17. Additional Forms & Databases can be added
